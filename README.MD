## Instalação:

```bash
# Baixar projeto
$ git clone git@github.com:caioagiani/blaze-bot.git

# Configurar .env
$ cp .env.example .env

# Instalar dependências
$ yarn install ou npm install

# Iniciar aplicação
$ yarn dev ou npm run dev
```
